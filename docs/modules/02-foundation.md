# Module 2: Foundation

!!! abstract "Time: 2 hours"

    Core concepts, terminology, and frameworks for benefit-risk assessment.

## Learning Objectives

By the end of this module, you will be able to:

1. **Define** key benefit-risk terminology used in regulatory and pharmacovigilance contexts
2. **Describe** the product lifecycle perspective on benefit-risk assessment
3. **Explain** the B-R capability maturity model levels
4. **Apply** the Structured Benefit-Risk Framework (SBRF) step-by-step process

## Key Takeaways

!!! success "Essential Points"

    1. **Benefit-risk assessment is a continuous process** that spans the entire product lifecycle from development through post-marketing
    2. **Structured frameworks** (like SBRF) improve transparency and consistency in B-R decision-making
    3. **Terminology matters** — using standardized terms enables clear communication across stakeholders
    4. **Maturity develops over time** — organizations progress through capability levels from ad-hoc to optimized B-R processes
