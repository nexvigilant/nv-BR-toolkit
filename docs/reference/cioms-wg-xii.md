# CIOMS Working Group XII

## Overview

The **Council for International Organizations of Medical Sciences (CIOMS) Working Group XII** developed comprehensive guidance on benefit-risk assessment in drug development.

## Key Recommendations

1. **Structured Framework** - Use systematic approaches
2. **Transparency** - Document decision rationale
3. **Stakeholder Input** - Include patient perspectives
4. **Lifecycle Approach** - Continuous reassessment
