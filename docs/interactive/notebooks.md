## Running Locally

```bash
# Clone the repository
git clone https://github.com/nexvigilant/nv-BR-toolkit.git
cd nv-BR-toolkit

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook notebooks/
```

## Requirements

- Python 3.10+
- pandas, numpy, scipy, matplotlib
- Jupyter Notebook
